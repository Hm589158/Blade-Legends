# أساطير النصل — Legends of the Blade

لعبة RPG للهاتف بـ **الوضع الأفقي**، عربية بالكامل (RTL)، بطابع «السبج المنصهر والذهب».
كل الشخصيات والوحوش مرسومة بـ **CSS خالص** — لا صور ولا ملفات صوت خارجية إطلاقاً.

> القوة = **العتاد فقط** (لا مهارات لاعب). تقاتل بـ **٣ أبطال** بأسلوب **تتابعي**،
> تصنع وتشتري **أطقماً كاملة**، وترقّيها.

---

## الفكرة الأساسية

- **العتاد طقم كامل مربوط بالبطل.** الطقم يحدّد *اسم* البطل، *عنصره*، *مظهره*، و*إحصاءاته*.
- **القوة من العتاد فقط:** ندرته × إحصاءاته × مستوى تحسينه (+N) × رتبة البطل.
- **٣ أبطال** تقاتل بهم بأسلوب تتابعي وترقّيهم.
- **عملتان:** 🪙 ذهب (من الزنزانات، للتصنيع والترقية) و 💎 جواهر (نادرة، للسوق فقط).

## التبويبات

1. **الزنزانات 🗺️** — قتال تتابعي: البطل الأول يقاتل، وإذا سقط دخل التالي. النصر يمنح ذهباً + مادة، و**أول اجتياز** لكل زنزانة يمنح **جواهر**.
2. **الفريق 🛡️** — ٣ أبطال (الترتيب #1/#2/#3 = ترتيب دخولهم). جهّز الأطقم، ورقِّ العتاد (+N) ورتبة البطل.
3. **التصنيع ⚒️** — اختر المستوى (برونزي→مقدّس) و**العنصر**، وادفع مواد + ذهب لصنع طقم.
4. **السوق 🏪** — متجر بالجواهر (انظر أدناه).

## نظام السوق 🏪

السوق عملته **الجواهر فقط** (أصعب من الذهب، فالجواهر نادرة)، وفيه قسمان:

**المتجر:**
- **عرض مميّز** — طقم أسطوري «محدود» بسعر جواهر مرتفع.
- **معروضات اليوم** — ٦ أطقم بمستويات مختلفة، سعرها بالجواهر حسب الندرة، يُشترى كلٌّ مرّة واحدة (ثم «نفد»).
- **تجدّد يومي مجاني** (حسب تاريخ الجهاز) + زر **تحديث فوري** مدفوع بالجواهر.
- **نافذة تأكيد شراء** بمعاينة الطقم (إحصاءاته وعنصره) قبل الدفع.

**التبادل:**
- **بدّل** فائض المواد/الذهب ← جواهر (⛓️١٢/🔷٦/💠٢ أو 🪙١٢٠٠ ← جواهر).
- **بِع** أطقمك الفائضة من المخزن مقابل جواهر (القيمة حسب الندرة والتحسين).

اقتصاد الجواهر مصمّم ليبقى «صعباً»: تأتي من أوّل اجتياز للزنزانات ومن استثمار مواردك في التبادل/البيع. الأطقم المشتراة تذهب للمخزن وتُجهّز من «الفريق».

## هوية العناصر

عجلة القوة: نار ⟶ هواء ⟶ أرض ⟶ ماء ⟶ نار. «روح» ×١٫٥ هجوماً ودفاعاً.
كل عنصر يمتاز (تبادل لا أفضلية): 🔥 نار هجوم · 💧 ماء قلوب · 🪨 أرض دفاع · 🌪️ هواء هجوم رشيق · ✦ روح متوازن.
المواد: ⛓️ حديد · 🔷 بلّورة · 💠 جوهر — الزنزانة الأصعب تمنح الأندر.

## البنية التقنية

```
asateer-al-nasl/
├─ index.html          # هيكل أفقي + روابط CDN (Vue 3 / GSAP / الخطوط)
├─ css/styles.css      # كل الأنماط + فنّ CSS (P6 redesign، P7 landscape، P8 market)
└─ js/
   ├─ constants.js     # العناصر+هوياتها، الندرة، المواد، التصنيع، الزنزانات، اقتصاد السوق
   ├─ gear.js          # توليد الأطقم (makeSet)، الإحصاءات، التحسين، التكاليف، قيمة البيع
   ├─ battle.js        # رياضيات القتال (computeDamage)
   ├─ audio.js         # مؤثرات Web Audio (بلا ملفات)
   ├─ persist.js       # الحفظ/التحميل في localStorage
   └─ app.js           # Vue: الحالة، القتال التتابعي، السوق، الحفظ التلقائي
```

- **التقنية:** Vue 3 + GSAP عبر CDN. خطوط Amiri + Cairo.
- **بلا أصول:** صفر صور/صوت.
- **الحفظ تلقائي** (يشمل مخزون السوق اليومي)؛ «تصفير الحفظ» في التصنيع (تأكيد بضغطتين).
- **إتاحة:** يحترم `prefers-reduced-motion`.

## نقطة تمديد للذكاء الاصطناعي

`makeSet(tier, element)` في `js/gear.js` (يستخدمها التصنيع والسوق) معزولة ومُعلَّمة بـ `SWAP POINT`؛
استبدلها بنداء توليدي يعيد **نفس الشكل** دون لمس بقية الكود.

## التشغيل

افتح `index.html` بمتصفّح (الأفضل بشاشة أفقية / هاتف بالعرض). يلزم إنترنت لتحميل مكتبات CDN والخطوط أوّل مرّة.
